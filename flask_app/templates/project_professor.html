Professor Rating Website

This project is a rudimentary professor rating website. Students could register accounts, then use those accounts to post reviews about professors. If a professor is not already in the system, students could submit requests to add new professors. The site uses a MongoDB database, and is currently hosted on heroku. Though the site can still be visited, it may take a few minutes for the heroku dyno to spinup.
